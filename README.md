# Pokedox-Kenneth Tenny


## Pre Requisites
1. A cross platform Web Server(such as XAMPP) which acts as a localhost for the application.

## Usage

0: Cloning the repo.

1: Start Apache Web Server

2: Shoot http://localhost/newgen/index.html on your web server.(If you are not using the default port, include the port number with a colon in the URL)


## Libraries, Frameworks and Programming Languages

1. AJAX
2. Bootstrap
3. JavaScript
